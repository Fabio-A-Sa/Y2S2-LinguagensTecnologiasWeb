# 0 - Web Introduction

### Internet vs. Web

#### 1. Internet
Computadores interligados numa rede a nível global usando um conjunto de protocolos

#### 2. Web
Sistema de documentos de hipertexto (links) acessados via internet / browser. Também conhecida como WWW (World Wide Web)
